Notebooks#

Tijdens deze course gebruiken we Google Colab als primaire manier om te oefenen met Python. Colab maakt het mogelijk om Python code gratis in de cloud te draaien zonder Python te hoeven installeren op je computer. Ook kun je met Google Colab gebruik maken van de beschikbare GPU‚Äôs.

Het document dat je leest is geen statische webpagina, maar een interactieve omgeving, ook wel notebook genoemd, waarmee je code kunt schrijven en uitvoeren. Notebooks bestaan ‚Äã‚Äãuit zogenaamde codecellen, blokken van √©√©n of meer Python-instructies. Hier is bijvoorbeeld een codecel die het resultaat van een berekening (het aantal seconden in een dag) opslaat in een variabele en het resultaat print:

seconden_in_een_dag = 24 * 60 * 60
print(seconden_in_een_dag)

86400

Klik op de ‚Äúplay‚Äù knop om de code uit te voeren (hover over de bovenstaande code, dan komt de knop tevoorschijn). Soms moet je even wachten, maar na een aantal seconden zou er bij de play knop een groen vinkje moeten verschijnen waarbij onder het blok code het resultaat van de berekening komt te staan. Ook is het mogelijk om de code uit te voeren met Ctrl + Enter (Command + Enter voor Mac gebruikers).

Variabele uit de ene cel kun je ook gebruiken in latere cellen, zoals hier:

seconden_in_een_week = 7 * seconden_in_een_dag
seconden_in_een_week

604800

Let op dat de volgorde van uitvoering van de cellen belangrijk is. Als we bijvoorbeeld de cel die seconden_in_een_dag opslaat niet van tevoren uitvoeren, zal de bovenstaande cel een fout genereren, omdat deze afhankelijk is van deze variabele. Om er zeker van te zijn dat alle cellen in de juiste volgorde uitvoert, kunt je ook klikken op ‚ÄúRuntime‚Äù in het hoofdmenu en vervolgens op ‚ÄúRun all‚Äù.

Oefening Voeg een cel toe onder deze cel: klik op deze cel en klik vervolgens op ‚Äú+ Code‚Äù (door net onder deze cel met je muis te hoveren). Bereken in de nieuwe cel het aantal seconden in een jaar door de variabele seconden_in_een_dag opnieuw te gebruiken. Voer de nieuwe cel uit.

Andere soorten data in Python#

Om nuttige dingen te kunnen doen met Python willen we vaak op andere manieren met data werken, waarbij alles in een variabele plaatsen niet overzichtelijk/effici√´nt/werkbaar is. Hier zullen we er een paar behandelen.

Uitleg#

Wat kunnen we doen als we niet 1 nummer willen opslaan, maar meerdere nummers? Die kunnen we plaatsen in een lijst (list).

#Als we een lijst met prijzen hebben van een product kunnen we die opslaan in een lijst.
#De lijst wordt aangegeven door blokhaken, waarbij de prijzen worden gescheiden door komma's.
prijzen = [10, 20, 15, 12]

#We kunnen een lijst ook eerst leeg aanmaken en daarna pas vullen.
kleuren = []
kleuren.append("blauw")
kleuren.append("rood")
kleuren.append("paars")
kleuren.append("oranje")

#Voer ook dit maar eens uit, als het goed is zie je twee lijsten.
print(prijzen)
print(kleuren)

['blauw', 'rood', 'paars', 'oranje']
